SHEET 1: DEFINITIONS
====================================================================================================

City                 | The city folder name (tweets mentioning this city)
Total Tweets         | Total number of tweets in this dataset
In This City         | User profile location matches this city (e.g., Baltimore, Buffalo)
In Other Cities      | User profile location is a different specific city/county (e.g., Los Angeles, Seattle)
Top 3 Other Cities   | The 3 most common other cities where users are located
Vague Info           | User profile location is too broad (e.g., USA, California, Texas)
No Info              | User profile has no location information at all
Data Quality         | Percentage of tweets with specific city-level location (In This City + In Other Cities)
